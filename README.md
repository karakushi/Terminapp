#Terminapp als Web API mit C# implementiert

Diese App ist eine Terminapp als Web API, welche mit C# programmiert wurde. Die App soll es Benutzern ermöglichen, Termine zu erstellen, zu bearbeiten, zu löschen und zu lesen. Dabei habe ich eine Schnittstelle erschaffen, die eine einfache und intuitive Verwaltung von Terminen ermöglicht.

In der Projektdokumentation ist meine Arbeit Schritt für Schritt dokumentiert und erklärt.
